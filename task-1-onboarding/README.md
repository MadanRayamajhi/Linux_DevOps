# Task 1: Developer Onboarding

## Quick Summary
Sets up a `developer` user with SSH access and `/opt/project_alpha` project directory.

## What It Does
1. Creates `developer` user
2. Creates `/opt/project_alpha` directory
3. Sets ownership to developer
4. Sets permissions to 700
5. Configures SSH keys

## Usage
```bash
sudo bash commands.sh
```
