# Task 5: Resource Investigator

## Quick Summary
Monitors and displays system resource usage including CPU, memory, disk space, and running processes.

## What It Does
1. Displays system uptime and load average
2. Shows CPU usage information
3. Displays memory and swap usage
4. Shows disk space usage for mounted filesystems
5. Lists top running processes by resource consumption

## Usage
```bash
bash command.sh
```
