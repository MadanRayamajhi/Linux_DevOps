# Task 3: Firewall and Web Security

## Quick Summary
Configures UFW (Uncomplicated Firewall) to secure the system while allowing SSH and web application traffic.

## What It Does
1. Checks current UFW firewall status
2. Opens port 22/tcp for SSH access
3. Enables the firewall
4. Opens port 8080/tcp for web applications
5. Tests outbound internet connectivity
